## 1. 이전 세션(세션 4) 결과 요약

### 완료 항목
- **⑦ RSI Wilder SMMA (`src/analyzer.py`)**
  - `_calc_rsi()`: SMA 기반 rolling mean → Wilder's SMMA (ewm com=period-1, adjust=False) 교체
  - `delta.clip(lower=0)` / `(-delta).clip(lower=0)` 사용으로 gain/loss 분리 단순화
  - avg_loss == 0 엣지케이스 유지 (RSI = 100.0)

- **⑧ analyze 보유종목 연동 (`main.py`)**
  - `cmd_analyze()`: `get_holdings()` 호출 → 종목코드 매칭 시 `avg_price` 실전값 전달
  - API 실패 시 `except Exception: pass`로 avg_price=0 폴백 처리
  - 보유종목이면 "보유중 - 평균매입가: NNN원" 출력 추가

- **⑨ history --date 옵션 (`main.py`)**
  - `cmd_history()`: `args.date` 우선, 없으면 오늘 날짜 사용
  - 파서: `--date` 인수 추가 (`default=None`, `metavar="YYYYMMDD"`)
  - 에러 메시지도 date_str 동적 출력으로 수정

### 변경 파일
- `src/analyzer.py` — `_calc_rsi()` 함수 (line 10-23)
- `main.py` — `cmd_analyze()`, `cmd_history()`, history 파서 (line 35-107)
